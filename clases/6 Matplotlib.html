Graphics with Matpletlib#

Lines, symbols and colors#

The simplest way to choose the curve graph mode is through a third argument. This argument, which appears immediately after the data (X and Y), allows to control the type of line or symbol used in graphing. In the case of the solid line, it can be specified with a script (‘-’) or simply not putting anything, since solid line is the default symbol. The two previous specifications are equivalent. You can also choose the color, or the symbol to be used with this argument:

To add names to the curves, we have to add a Label, in this case in the sameplot ()command, and then Show it with `legend ()

Two graphics in the same figure#

There are several functions that allow to put more than one graph in the same figure. Let’s look at an example using Figure and add_sublot

graf2 = fig_1.add_subplot(122) # 1: number of rows 2: number of columns 2: the number of subploot

graf1.plot(x,y,linewidth=5, linestyle='dashed', label='Teoría')

Here can also be used latex type format for part of the text. If we use an expression enclosed among the $ symbols, Matpletlib interprets that it is written in (a subset) of Latex.

** Matpletlib ** has an internal symbols processor to show the latex notation that recognizes the most common elements, or it can be chosen to use an external latex processor.

plt.plot(x,y,linewidth=5, linestyle='dashed', label=r"$\frac{\sin(e^2)}{[2 + \cos (x)]^2}$")
